# Challenge Summary
Implement your own Graph.

## Challenge Description
The graph should be represented as an adjacency list, and should include the following methods:

* add_node()
* add_edge()
* get_node()
* get_neighbors()
* size()

## Approach & Efficiency
* add_node()
    * Big O speace O(1)
    * Big O time O(1)
* add_edge()
    * Big O speace O(1)
    * Big O time O(1)
* get_node()
    * Big O speace O(1)
    * Big O time O(1)
* get_neighbors()
    * Big O speace O(1)
    * Big O time O(1)
* size()
    * Big O speace O(1)
    * Big O time O(1)
