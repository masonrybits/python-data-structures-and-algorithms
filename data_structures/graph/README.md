# Challenge Summary
Implement your own Graph.

## Challenge Description
The graph should be represented as an adjacency list, and should include the following methods:

* add_node()
* add_edge()
* get_nodes()
* get_neighbors()
* size()

## Approach & Efficiency
* add_node()
    * Big O speace O(1)
    * Big O time O(1)
* add_edge()
    * Big O speace O(1)
    * Big O time O(1)
* get_nodes()
    * Big O speace O(1)
    * Big O time O(1)
* get_neighbors()
    * Big O speace O(1)
    * Big O time O(1)
* size()
    * Big O speace O(1)
    * Big O time O(1)

## API
* add_node()
    * input: a value
    * output: node with the value
* add_edge()
    * input: 2 verrices, weight
    * output: None
* get_nodes()
    * input: None
    * output: a list of all vertices
* get_neighbors()
    * input: a vertex
    * output: list of tuples containing neighbors verices
* size()
    * input: None
    * output: number of vertices
